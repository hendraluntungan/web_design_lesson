HTML

HTML adalah Singkatan dari Hyper Text Markup Language adalah kerangka dari sebuah website. Struktur dari HTML terbagi menjadi 2 bagian, Head dan Body. Head berisi informasi seperti title, author, description dan meta. Tidak semua bagian head dimasukkan dalam tampilan web, tapi sangat penting bagi kebutuhan search engine. Sedangkan body akan berisi konten yang ada dalam website kita.

Struktur HTML

Struktur HTML : Doctypye, tag HTML, didalam tag HTML terdapat tag Head dan tag Body

HTML Elemen

Ketika tag sudah memiliki atribut, kemudian ada content, itu kita sebut HTML element. HTML element terdiri dari 2 jenis:

  • HTML element yang mempunyai opening tag dan closing tag. Contoh: tag p
  • Empty HTML element, hanya memiliki opening tag dengan garis miring sebelum kurung tutup. Contoh: tag br dan tag img

HTML Atribut

Atribut dalam HTML adalah properti tambahan dalam sebuah elemen untuk memberikan nilai-nilai tertentu. Setiap elemen dalam HTML punya global atribut atau atribut-atribut standar. Contoh: acceskey, class, id, dir, lang, style, tabindex dan title. Contoh: tag img, yang memiliki 2 atribut:

  • width = untuk menentukan lebar elemen gambar. Contoh: 80%.
  • src = untuk menentukan sumber dari gambar entah dari internet atau dari sumber lainnya.

HTML Comment

Fungsi dari comment dalam HTML adalah untuk memberikan komentar didalam code HTML kita, dan komentar ini tidak akan ditampilkan didalam website.
